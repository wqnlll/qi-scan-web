# Qi Scan Web

AI-powered scanning tools for the modern web.

## Features
- Website Scanner: Analyze SEO, speed, mobile, security
- Brand Scanner: Social presence, mentions, sentiment
- Product Scanner: Conversion optimization, UX analysis

## Quick Start
1. Clone or download this repository
2. Open `index.html` in your browser
3. Deploy to GitHub Pages, Netlify, or Vercel
